# eliud_generator

Eliud Code Generator

## Getting Started

Add these dev_dependencies to your project
  build_runner: "1.7.2"
  eliud_generator:
    path: ../eliud_generator

Then run flutter packages pub run build_runner build --delete-conflicting-outputs
